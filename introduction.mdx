## Use Cases

With Inkless's APIs for e-signing, you can:

- Send sales contracts programatically via CRMs like Salesforce or Hubspot
- Send onboarding agreements for new SaaS customers via Inkless's API
- Send purchase agreements programatically to a large number of customers

We allow you to easily and programatically send PDF template contracts for signing at scale via API\!
